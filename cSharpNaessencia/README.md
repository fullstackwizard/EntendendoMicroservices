?? Básico
Hello World: Crie um programa que exiba "Hello, World!" no console. ok
Operações matemáticas: Peça dois números ao usuário e exiba a soma, subtração, multiplicação e divisão. ok
Par ou Ímpar: Solicite um número e informe se ele é par ou ímpar.
Tabuada: Receba um número e exiba a tabuada dele até o 10.
Contador de vogais: Peça uma palavra e conte quantas vogais ela tem.


?? Intermediário
Calculadora: Crie uma calculadora que recebe dois números e uma operação (+, -, *, /).
Fibonacci: Exiba os primeiros N números da sequência de Fibonacci.
Número Primo: Verifique se um número é primo.
Jogo da Adivinhação: Gere um número aleatório entre 1 e 100 e peça para o usuário adivinhar.
Ordenação: Implemente o Bubble Sort para ordenar um array de números.


?? Avançado
CRUD Simples (Console App): Crie um CRUD de usuários utilizando listas e manipulação de arquivos.
Sistema de Login: Implemente um sistema de login e senha usando armazenamento em JSON ou banco de dados SQLite.
Consumindo API: Faça uma requisição a uma API pública e exiba os dados no console.
Threads e Tasks: Crie um programa que simule múltiplas tarefas rodando paralelamente.
Sistema de Gestão de Estoque: Desenvolva um sistema básico para cadastrar produtos, atualizar estoque e registrar vendas.
